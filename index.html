cv-segmentation-sam-unet

Веб-интерфейс для CV-сегментации игрового поля с помощью SAM и U-Net

Автоматическая сегментация площадки на спортивных видео с использованием передовых моделей компьютерного зрения: SAM (Segment Anything Model) и U-Net.

TensorFlow/numpy OpenCV segment-anything flask

sentiment-analyzer

Веб-интерфейс на базе Streamlit для анализа тональности текста

Система использует нейросетевые модели для определения эмоциональной окраски введённого текста (позитивная, негативная, нейтральная).

BERT KMeans Streamlit Scikit-learn
GitHub Демо (PDF)

realtime_object

Приложение для Android для детекции объектов в реальном времени

Приложение использует модель TensorFlow Lite для распознавания объектов через камеру в реальном времени и при обработке видео из галереи. Поддерживает экспорт результатов и работу без интернета.
